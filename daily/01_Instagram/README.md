# OneTeam

## Goal

- 각자 기여할 수 있는 부분에 꾸준히 기여를 해보자.
- 서로 부족한 부분, 혹은 발전될 부분을 도와주자.
- **모두 빠짐없이** 프로젝트에 자신의 흔적을 남기자.
- 첫 번째 릴리즈 버전 배포
	- 릴리즈 버전을 기반으로  새로운 모듈 생성 및 npm 업로드

## 사전지식

- Git / Github / Markdown
- Node.js / npm
- JavaScript : [Airbnb/JavaScript](https://github.com/tipjs/javascript-style-guide)
